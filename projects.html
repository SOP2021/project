Heat Exchanger Network Analysis

We have made a code that performs pinch analysis, by inputing stream data avaible in the problem system we can get the minimum hot and cold utility required by the system, where the pinch lies and construct the corresponding composite curves which can help with further analysis of finding the minimum number of heat exchangers necessary for building the heat exchanger network.

Adsorption Treatment of Waste Water

The project is related to build an interface in order to allow user to select a material as per his/her requirement. To achieve this objectives, we performed an extensive literature survey and with the help of a well-planned strategy, a handful data points were shortlisted. The selected data not only represented the overall purpose but also allowed us to give a solution with confidence. The rigorous exercise is expected to create a unique opportunity that can be further scaled at multiple levels based on the techno-economic feasibility aspects..
